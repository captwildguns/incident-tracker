Feature Documentation

In-depth documentation of all application features and capabilities

Dashboard

Purpose

The Dashboard provides a comprehensive overview of incident activity, trends, and key metrics for safety coordinators and administrators.

Key Statistics Cards

  1. Total Incidents: Complete count of all incidents in the system
  2. Active Incidents: Number of incidents currently open or under review
  3. Critical Incidents: High-priority incidents requiring immediate attention
  4. Avg Response Time: Average time from incident creation to first action

Incident Trends Chart

Type: Line Chart (Recharts)

  • X-Axis: Time period (months)
  • Y-Axis: Number of incidents
  • Color: Brand blue (#4A6FA5)
  • Interactive hover tooltips

Incidents by Type (Pie Chart)

Visual breakdown of incident categories:

  • Student Behavior
  • Safety Violation
  • Mechanical
  • Accident
  • Other

Recent Incidents Timeline

Shows the 5 most recent incidents with clickable Incident IDs that open detailed views in a modal dialog.

Incidents Management

Advanced Filtering System

  • Search Bar: Real-time search across incident ID, student name, driver name, description
  • Status Filter: All, Open, Under Review, Resolved, Closed
  • Severity Filter: All, Critical, High, Medium, Low
  • Type Filter: All, Student Behavior, Safety Violation, Mechanical, Accident, Other

Sortable Columns

All 8 columns support sorting:

  • Incident ID
  • Date (default sort: descending)
  • Student
  • Driver
  • Vehicle
  • Type
  • Severity (custom order: Critical → High → Medium → Low)
  • Status (custom order: Open → Under Review → Resolved → Closed)

Visual Indicators:

  • ⇅ Double chevron: Sortable but not active
  • ↑ Up chevron: Sorted ascending
  • ↓ Down chevron: Sorted descending

Export Functionality

Export filtered incident list to CSV format with all visible columns included.

Students Management

Student Table with Photos

Visual Features

  • Student Photos: 40px circular avatars with professional photos or initials
  • Active Incident Badge: Orange "Active" badge appears next to name for students with open incidents
  • Grade Filter: Filter by specific grade level (K-12)
  • Default Sort: By Name (ascending)

Student Detail View

Click Student ID to view comprehensive profile including:

  • Contact information
  • Transportation details (run, pickup/dropoff)
  • Medical information
  • Complete incident history

Fleet Management (Vehicles)

Vehicle Table with Bus Images

Visual Features

  • Bus Photos: 40px circular images showing actual bus models
    • Blue Bird Vision
    • IC Bus CE Series
    • Thomas Saf-T-Liner C2
    • Blue Bird All American
    • Thomas Saf-T-Liner HDX
  • Maintenance Icons:
    • ✓ Checkmark: Excellent/Good
    • ⚠ Warning: Needs Attention
    • 🔧 Wrench: In Repair
  • Trend Indicators:
    • ↑ Red: High incidents (>8)

Vehicle Detail View

Comprehensive vehicle information including:

  • Full-size bus photo and specifications
  • GPS & AVL configuration
  • Garage assignments
  • Maintenance & inspection schedule
  • Complete incident history

Driver Management

Driver Table with Photos

Visual Features

  • Driver Photos: 40px circular avatars with professional photos
  • Employee ID: Displayed below name in smaller text
  • Safety Rating: Star icon with rating out of 5.0
  • Default Sort: By Name (ascending)

Driver Detail View

Complete driver profile including:

  • Employment information and years of service
  • Certifications & licenses (CDL, First Aid, Training)
  • Assignment information (vehicle, runs)
  • Safety & performance metrics
  • Complete incident history
  • Communication history

New Incident Entry

5-Step Wizard

  1. Basic Information: Date, time, type, severity, location
  2. Involved Parties: Student(s), driver, vehicle
  3. Incident Details: Description, actions taken, injuries, witnesses
  4. Additional Information: Attachments, follow-up, notes, tags
  5. Review & Submit: Review all information and submit or save as draft

Auto-Save: Form automatically saves every 2 minutes to prevent data loss.

Reports & Analytics

Quick Reports

  1. Incident Summary Report: Total incidents by type, severity, status
  2. Driver Performance Report: Individual driver statistics and safety ratings
  3. Vehicle Maintenance Report: Maintenance status and upcoming inspections
  4. Student Incident Report: Students with multiple incidents and patterns
  5. Compliance Report: State and federal reporting requirements

Custom Report Builder

6-step process to create custom reports:

  1. Select Data Source (Incidents, Students, Drivers, Vehicles)
  2. Choose Columns to include
  3. Apply Filters with complex logic
  4. Set Sorting & Grouping options
  5. Add Visualization (charts)
  6. Preview & Export (PDF, Excel, CSV)

Driver Communications

Communication Hub Features

  • Track all driver communications in one place
  • Message threading with complete conversation history
  • Communication status tracking (Pending, Responded, Acknowledged)
  • Filter by driver, incident, date range, status, priority
  • Templates for common communications
  • Complete audit trail for compliance

Communication Types

  • Email
  • Phone
  • In-Person
  • Text Message

Search & Filtering

Universal Search Features

  • Real-time filtering: Results update as you type
  • Case-insensitive: Matches regardless of case
  • Partial matches: Finds results containing search term
  • Multi-field search: Searches across multiple columns

Advanced Filtering

All data pages support:

  • Search bar for text searches
  • Dropdown filters for categories
  • Date range filters
  • Multiple filters can be combined
  • Clear all filters button

Sorting Functionality

Sort Behavior

  1. First Click: Sort ascending (A→Z, 0→9, oldest→newest)
  2. Second Click: Sort descending (Z→A, 9→0, newest→oldest)

Visual Indicators

  • ⇅ Double chevron (gray): Column is sortable but not currently sorted
  • ↑ Up chevron (black): Sorted ascending
  • ↓ Down chevron (black): Sorted descending

Default Sorts by Page

Page Default Column Direction
Dashboard Date Descending
Incidents Date Descending
Students Name Ascending
Vehicles Details (Name) Ascending
Drivers Name Ascending
